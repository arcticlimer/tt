# tt 

tt is a simple program that helps users to apply the [20-20-20 rule](https://www.healthline.com/health/eye-health/20-20-20-rule).

# Using it
## Download
```bash
git clone https://github.com/arcticlimer/tt && cd tt
```

## Running it

### With nix
```bash
nix run

```
### With cargo
```bash
cargo run
```
